# Screenshots

Screenshots will go here. To add yours:

1. Launch the app
2. Press Cmd+Shift+4 (macOS) or Snipping Tool (Windows)
3. Save images to this directory as:
   - `main_window.png` — Main TTS generation interface
   - `voice_clone.png` — Voice cloning with waveform
   - `speakers.png` — Speaker picklist with descriptions
